# Valentine's Day Card

Live: https://mrishav.github.io/valentines/

An interactive Valentine's Day card built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no build step — just open `index.html` in a browser.

## What it does

- Envelope opens to reveal the card
- Asks a question with a "No" button that runs away when you hover it
- Clicking "Yes" triggers a confetti celebration
- Floating pixel hearts you can click to pop

## Stack

- HTML, CSS, vanilla JS
